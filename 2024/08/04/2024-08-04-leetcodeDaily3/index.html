572. 另一棵树的子树

经典题目

思路

判断是否是一棵树的子树，那么我们首先想到如何判断两个树是相同的。

判断两个树是否相同有五种情况：

  • 左右子树为空：true
  • 左子树为空，右子树不为空：false
  • 左子树不为空，右子树为空：false
  • 左右子树都不为空，但左右节点值不相等：false
  • 左右子树都不为空，且左右节点值相等：true
if (node1 == null && node2 == null) return true;
if (node1 == null || node2 == null || node1.val != node2.val) return false;

于是我们递归的判断左子树和右子树

return isTree(node1.left, node2.left) && isTree(node1.right, node2.right);

我们现在知道了如何判断两个树是相同的，于是我们开始判断是否为子树：

  • 首先判断当前节点时候匹配
if (root == null) return false;
if (isTree(root, subRoot)) return true;
  • 如果不匹配再去遍历左孩子和右孩子：
return isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot);

代码

public boolean isSubtree(TreeNode root, TreeNode subRoot) {
     if (root == null) return false;
     if (isTree(root, subRoot)) return true;

     return isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot);
 }

 private boolean isTree(TreeNode node1, TreeNode node2) {
     if (node1 == null && node2 == null) return true;
     if (node1 == null || node2 == null || node1.val != node2.val) return false;

     return isTree(node1.left, node2.left) && isTree(node1.right, node2.right);
 }
